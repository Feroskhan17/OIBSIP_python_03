# OIBSIP_python_03
The BMI Calculator is a Python-based project developed as part of Task 1 of the Oasis Infobyte Internship. It calculates Body Mass Index using user-provided height and weight and classifies the result into standard health categories. This project helps beginners practice Python basics and logic.
The Password Generator is a Python-based project that generates strong and secure random passwords. It allows users to create passwords using a mix of uppercase letters, lowercase letters, numbers, and special characters. This project is ideal for beginners to understand Python fundamentals and secure password practices.
✨ Features
Generates random and strong passwords
User-defined password length
Uses uppercase, lowercase, digits, and special characters
Simple and beginner-friendly Python code
🛠️ Technologies & Concepts Used
Python 3.x
random module
string module
Loops and conditional statements
User input handling
